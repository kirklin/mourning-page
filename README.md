# mourning-page
哀悼日网页变灰

## Usage

### Way one
```javascript
import renderGrayPage from "mourning-page"
renderGrayPage();
```
```javascript
import renderGrayPage from "mourning-page"
renderGrayPage(null,"CSS_FILTER",100);
```
### Way two
```javascript
import renderGrayPage from "mourning-page"
renderGrayPage(null,"CSS_MIX_BLEND_MODE")
```

### Way three
```javascript
import renderGrayPage from "mourning-page"
renderGrayPage(null,"CSS_BACKDROP_FILTER")
```

### Install

#### pnpm
```bash
pnpm i mourning-page
```

#### npm
```bash
npm i mourning-page
```
